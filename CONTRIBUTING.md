# Contributing

This is a research planner repository.

## Before you contribute

1. Read [`README.md`](README.md) for current scope and status.
2. Read [`plans/00-master-plan.md`](plans/00-master-plan.md) for the master plan.

## Rules

- No illegal packages (license-incompatible or pirated dependencies).
- Use Conventional Commits where reasonable.

## Code of Conduct

This repository follows the [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/). Be excellent to each other.

## Questions

Open a GitHub Discussion or an Issue.
